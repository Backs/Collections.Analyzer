# CI0009: List capacity should be set based on the source collection

When creating a `List` that is immediately populated from another collection in a loop, it is more efficient to initialize the `List` with a capacity equal to the source collection's size. This avoids multiple internal array resizes as elements are added.

## Examples

### Violation
```csharp
public List<int> Method(int[] array)
{
    var list = new List<int>();
    foreach (var item in array)
    {
        list.Add(item * 2);
    }
    return list;
}
```

### Fix
```csharp
public List<int> Method(int[] array)
{
    var list = new List<int>(array.Length);
    foreach (var item in array)
    {
        list.Add(item * 2);
    }
    return list;
}
```
